# vim操作
```
进入vim后按i进入插入模式
:q 退出
:wq 保存退出
:w 保存
:w! 强制保存
:q! 强制退出
:wq! 强制保存退出
G 跳转到最后一行
gg 第一行的第一个字符

```

# 杀进程

```
kill id:杀死进程
kill -KILL id:强制杀死进程
kill -9 id:彻底杀死进程
kill -u username :杀死指定用户所有进程
```
