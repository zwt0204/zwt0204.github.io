# 远程连接
1. 依次选择Tools-Deployment-Configuration

2. 然后进入如下图所示界面：依次点击+，SFTP，弹窗出来起名字，随便起就行。

3. 接着就是配置画面，有Connection、Mapping、Exclude Paths，下图是Connection的配置，主要配置项目和服务器的连接。

4. 然后配置Mapping。

5. 配置Exclude Path

6. 配置解释器，Files → Setting → Protect → python interpreter
