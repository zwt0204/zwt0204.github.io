# 基本概念
传统的卷积神经网路只能处理欧式空间的数据，因为这些领域的数据具有平移不变性。平移不变性的优点使得我们可以再输入数据空间定义全局共享的卷积核，从而定义卷积神经网络。但是在图数据中，不存在平移变性，所以这对图数据上的卷积神经网络的定义提出了挑战。

主要的挑战：
```
图数据是非欧式空间
图数据具有多样的特性
图数据的规模很大
```
